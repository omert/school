\begin{enumerate}
  \item 
    Let $H$ be any non-planar graph. Let $G$ be the n-by-n grid. Then $G$
    does not contain $H$, since $G$ {\em is} planar, and the family of planar
    graphs is closed with respect to the operation of taking a minor. 
    Since we showed that
    the tree-width of $G$ is $O(\sqrt{n})$, it is not bounded.
  \item
    An alternative condition for $H$ being a minor of $G$ is that there exists
    a function $f$ from the nodes of $G$ onto the nodes of $H$ so that
    if there is an edges between $f(v)$ and $f(u)$ then there is an edge
    between $v$ and $u$.
    
    Let $G$ be a graph, and let $T_G$ be a minimal width tree decomposition of
    it. Let $H$ be a minor of $G$, and $f$ a function satisfying the condition
    above. Let $T_H$ be constructed from $T_G$ by applying $f$ to each of 
    the nodes in the {\em bags} which label the nodes of $T_G$. Then $T_H$
    is a tree decomposition of $H$: 
    \begin{itemize}
    \item
      If $e$ is an edge between $f(u)$ and $f(v)$
      then $u$ and $v$ would be in some bag of $T_G$, and hence $f(u)$ and
      $f(v)$ would be in some bag of $T_H$. 
    \item
      Since all the bags that
      contain $u$ are a connected component in $T_G$, then all the bags that
      contain $f(u)$ are a connected component in $T_H$.
    \end{itemize}
    
    We've thus constructed a tree decomposition for $H$. Since this 
    decomposition is not necessarily optimal, we've shown that the tree width
    of $H$ is at most the tree width of $G$.

\end{enumerate}
